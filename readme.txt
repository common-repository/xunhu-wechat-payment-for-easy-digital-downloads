== Installation ==
1. 先安装WooCommerce插件
2. 安装 "xunhu-wechat-payment-for-WooCommerce" 插件并激活
3. 在<a href="https://pay.xunhuweb.com" target="_blank">迅虎支付</a>注册账号
4. 绑定收款微信账号
5. 申请开通微信H5支付，获取ID和密钥
6. 在插件设置中填入ID和密钥
